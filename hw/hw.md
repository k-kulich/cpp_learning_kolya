## ДЗ по презе 1

Здесь будет сразу несколько домашних задач, но все они нужны для тренировки твоих навыков и пройденного материала.

Итак, задачи:

1. Написать функцию, которая принимает два целых числа и возвращает их среднее арифметическое.

2. Написать программу, которая читает из стандартного ввода целое число N, а затем выводит числа от 1 до N через пробел, но вместо чисел, кратных 3, выводит "Fizz", вместо кратных 5 – "Buzz", а вместо кратных 3 и 5 – "FizzBuzz".

3. Реализовать рекурсивную функцию вычисления факториала и итеративную версию. Сравнить производительность для больших чисел (не в коде, а на словах).

4. Написать программу, которая принимает из командной строки два аргумента (строки) и выводит их конкатенацию.

## ДЗ по презе 2

Задачи по мере возможностей можно делать, можно не делать. Будет круто, если попробуешь.

1. Объявить массив из 5 целых чисел. С помощью указателя пройти по массиву и вывести все элементы.

2. Написать функцию `void reverse(int* arr, size_t size)`, которая разворачивает массив на месте, используя указатели. (не возвращает новый массив, а разворачивает существующий).

3. Написать функцию `size_t my_strlen(const char* str)`, которая вычисляет длину строки без использования стандартных функций (для C-Style строк).

4. Написать программу, которая выводит все переданные аргументы (в командную строку - то есть те, которые при запуске программы вводятся), но в обратном порядке (от `argv[argc-1]` до `argv[0]`).

5. Написать функцию, которая принимает массив целых и функцию-компаратор `(bool cmp(int a, int b))` и возвращает указатель на максимальный элемент (аналогично примеру из лекции).
